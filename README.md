# 🚗 Automobile Data Analysis Project

## 📌 Objective
This project focuses on analyzing automobile data using Python libraries to understand how vehicle attributes affect pricing, performance, and market trends.

---

## 🛠️ Tools & Technologies Used
- Python 🐍  
- Pandas 📊  
- Matplotlib 📈  
- Seaborn 🎨  
- Jupyter Notebook 📓  

---

## 📂 Dataset Information
The dataset contains automobile features such as:
- Make / Brand  
- Engine Size  
- Horsepower  
- Fuel Type  
- Body Style  
- Price  
- Mileage (MPG)

---

## 📊 Key Visualizations Performed
- Correlation Heatmap (Price vs Features)
- Bar Chart (Average Price by Brand)
- Boxplot (Price Distribution by Body Style)
- Scatter Plot (Horsepower vs Price)
- Scatter Plot (Engine Size vs MPG)
- Pairplot (Multivariate Analysis)

---

## 🔍 Key Insights
- Price increases with engine size and horsepower  
- Luxury brands have significantly higher prices  
- Larger engines reduce fuel efficiency  
- Body style affects price variation  

---

## 📌 Conclusion
This analysis helps understand how technical and design features influence automobile pricing and performance. Visualization techniques made hidden patterns easy to interpret.
